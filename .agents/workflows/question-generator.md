# 문제 생성 에이전트 (Question Generator Agent)

재산보험 지식 베이스(`knowledge_base/` 폴더)를 바탕으로 시험 문제를 자동 생성합니다.
이 에이전트는 **"보상하는것은?"**, **"보상하지않는것은?"**, **"옳은것은?"**, **"옳지않은것은?"** 4가지 유형의 문제 생성에 특화되어 있습니다.

## 전제 조건

- `knowledge_base/` 폴더에 `.txt` 파일이 있어야 합니다.
- 파일 형식: `[주제명]` 헤더 아래에 한 줄에 하나의 사실(Fact)을 기술합니다.
- Python 3.x 환경이 필요합니다.

## 문제 유형 (4가지)

### 1. "보상하는 것은?" 유형
- **정답**: "보상한다", "담보한다" 등이 포함된 문장
- **오답**: "보상하지 않는다", "면책이다", "제외된다" 등이 포함된 문장 3개

### 2. "보상하지 않는 것은?" 유형
- **정답**: "보상하지 않는다", "면책이다" 등이 포함된 문장
- **오답**: "보상한다", "담보한다" 등이 포함된 문장 3개

### 3. "옳은 것은?" 유형
- **정답**: 해당 주제의 올바른 사실 문장
- **오답**: 다른 주제의 사실 문장 3개 (이 주제에 대한 설명으로는 부적절)

### 4. "옳지 않은 것은?" 유형
- **정답**: 다른 주제의 사실 문장 1개 (이 주제에 대한 설명으로는 부적절)
- **오답**: 해당 주제의 올바른 사실 문장 3개

## 실행 방법

// turbo
1. 프로젝트 루트에서 다음 명령어를 실행합니다:
```bash
python generate_ox_questions.py
```

2. 실행 후 `questions_ox.json` 파일이 생성됩니다.

## 설정 변경

`generate_ox_questions.py` 파일 상단의 설정을 수정하여 커스터마이징할 수 있습니다:

```python
OUTPUT_FILE = "questions_ox.json"   # 출력 파일명
NUM_QUESTIONS = 100                  # 생성할 문제 수
KB_DIR = "knowledge_base"            # 지식 베이스 디렉토리
```

## 문제 유형 비율

기본적으로 4가지 유형이 균등하게(각 25%) 생성됩니다. 비율을 변경하려면 스크립트의 `QUESTION_TYPE_WEIGHTS` 딕셔너리를 수정하세요.

## 출력 JSON 형식

```json
{
  "id": "ox-comp-001",
  "concept": "화재보험 특별약관 - 도난위험담보",
  "question": "다음 중 '도난위험담보 특별약관'에서 보상하는 것은?",
  "options": ["보기1", "보기2", "보기3", "보기4"],
  "answer": 2,
  "explanation": "정답은 2번입니다. ..."
}
```

## 지식 베이스 추가 방법

1. `knowledge_base/` 폴더에 새로운 `.txt` 파일을 추가합니다.
2. `[주제명]` 형식으로 주제를 구분합니다.
3. 각 줄에 하나의 사실을 작성합니다.
4. "보상한다/보상하지 않는다" 형태의 문장은 자동으로 보상/비보상으로 분류됩니다.

```text
[새로운 주제]
이 보험은 화재로 인한 손해를 보상한다.
지진으로 인한 손해는 보상하지 않는다.
계약자의 고의에 의한 손해는 면책이다.
```
