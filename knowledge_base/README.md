# 사용자 지식 베이스 (Knowledge Base) 가이드

이 문제 생성기는 사용자가 제공한 요약 자료를 바탕으로 문제를 생성할 수 있도록 설계되었습니다.
본인이 정리한 핵심 요약 노트나 교재 내용을 텍스트 파일로 저장해 두시면, AI가 이를 분석하여 문제를 출제합니다.

## 사용 방법

1. `knowledge_base` 폴더 안에 `.txt` 파일을 만드세요. (예: `concept_1_2.txt`)
2. 각 줄에 하나의 **"올바른 사실(Fact)"**을 적어주세요.
3. 주제를 구분하고 싶다면 `[주제명]` 형태로 헤더를 달아주세요.

## 작성 예시

```text
[계약 후 알릴 의무]
보험기간 중에 사고발생의 위험이 현저하게 변경 또는 증가된 사실을 안 때에는 지체 없이 보험자에게 통지해야 한다.
통지의무를 이행하지 않은 경우 보험자는 그 사실을 안 날로부터 1월 이내에 계약을 해지할 수 있다.
위험의 변경이 보험자의 승인 하에 이루어진 경우에는 통지의무가 면제된다.

[기업휴지손해]
기업휴지보험은 사고로 인해 영업이 중단됨으로써 발생한 상실이익과 고정비를 보상한다.
보상기간(Indemnity Period)은 복구에 소요되는 합리적인 기간을 의미한다.
```

## 주의사항
* 문장은 명확한 "사실" 위주로 작성해주세요.
* "~할 수 있다", "~해야 한다" 등 어미를 완결된 문장으로 써주세요.
* AI는 이 문장들을 정답 지문으로 사용하거나, 변형하여 오답 지문을 만듭니다.
